MODULE 01
基础认知 — 从天空开始理解无人机
建立对无人机系统的整体认知，掌握飞行原理与安全法规，为后续实操奠定理论基础
01
无人机概述与行业图谱
📺 25分钟 🎯 理论 📁 基础知识
无人机分类 行业全景 发展趋势
→
核心知识点
  • 多旋翼/固定翼/垂直起降无人机的结构与适用场景
  • 军用/民用/消费级无人机的等级划分与法规体系
  • 无人机产业链上中下游全景图谱
  • 低空经济政策背景与市场前景分析
  • 国内外主流无人机品牌与产品矩阵
  • 无人机驾驶员执照分类与考证路径
🎬 视频要点：通过行业报告数据与政策文件，立体呈现无人机赛道的机会与门槛
02
飞行原理与空气动力学
📺 25分钟 🎯 理论 📁 核心原理
飞行力学 升力原理 动力学
→
核心知识点
  • 旋翼产生的升力原理与推力控制机制
  • 无人机的六自由度运动模型（俯仰/横滚/偏航/升降）
  • 电机、螺旋桨、电池的性能匹配关系
  • 风速、风向对飞行稳定性的影响分析
  • 悬停状态的受力平衡与能效最优点
  • 无人机极限飞行包络线与安全边界
🎬 视频要点：配合飞行模拟器演示，让抽象的飞行力学可视化呈现
03
无人机系统组成与硬件架构
📺 25分钟 🎯 理论+认知 📁 硬件基础
飞控系统 动力系统 通信链路 任务载荷
→
核心知识点
  • 飞控系统：陀螺仪、加速度计、磁力计、气压计的作用
  • 动力系统：电机、电调、螺旋桨、电池的选型逻辑
  • 通信链路：遥控器、图传、数传链路的工作频段与距离
  • 任务载荷：可见光相机、红外热成像、激光雷达、多光谱等
  • 地面站系统（GCS）功能与软件界面认知
  • 整机参数表解读与主流机型横向对比
🎬 视频要点：以实机拆解视频配合动画，逐一拆解各子系统的工作逻辑
MODULE 02
组装实操 — 从零件到飞行
亲手完成无人机整机装配与调试，建立对硬件的直观认知，具备独立完成飞行前检查的能力
04
无人机硬件组装实操
📺 25分钟 🎯 动手实操 📁 组装流程
整机装配 布线焊接 实名登记
→
核心知识点
  • 机身结构件、机臂、起落架的安装顺序与注意事项
  • 电机安装与校准（正反转、动平衡检查）
  • 电调与飞控的接线逻辑（信号线、电源线定义）
  • 图传系统与天线的布线与屏蔽处理
  • 无人机实名登记与UOM平台账号注册
  • 组装完成后的外观检查与螺丝紧固复核
🎬 视频要点：全程实拍分步教学，标注每一步的关键细节与常见错误
05
飞控系统调试与地面站使用
📺 25分钟 🎯 动手实操 📁 飞控调试
飞控调参 地面站 固件升级
→
核心知识点
  • 飞控固件烧录与品牌地面站软件安装（QGroundControl / Mission Planner）
  • 遥控器对码、通道校准、模式切换逻辑
  • PID参数含义与调参基本原则（不要乱调！先理解再动手）
  • 传感器校准：加速度计、罗盘、气压计的校准流程
  • 飞行模式切换与失效保护（Failsafe）设置
  • 地面站地图加载、航点规划、航迹显示功能
🎬 视频要点：重点讲解调试参数的含义，避免学员死记硬背参数值
06
飞行安全规范与法规
📺 25分钟 🎯 法规+安全 📁 合规飞行
民用航空法 飞行禁区 保险
→
核心知识点
  • 《民用航空法》无人机章节与《无人驾驶航空器飞行管理暂行条例》
  • 实名登记、飞行资质、空域申请三件事的法律效力
  • 禁飞区、限飞区、管控区的识别与规避（含真实事故案例）
  • 视距内（VLOS）与超视距（BVLOS）飞行的条件与许可
  • 飞行前检查清单（PREFLIGHT CHECKLIST）标准化流程
  • 无人机第三者责任险的必要性解读
🎬 视频要点：结合真实违规处罚案例，强化法规意识与安全底线思维
MODULE 03
基础应用 — 从拍摄到数据
掌握航拍、影像处理、建模三大基础应用技能，建立"飞行-采集-处理"的全链路闭环
07
航空摄影测量基础
📺 25分钟 🎯 技能 📁 航测入门
像控点布设 航线规划 航向重叠率
→
核心知识点
  • 航空摄影测量的基本原理与坐标系转换
  • 像控点（Ground Control Point, GCP）布设方法与数量计算
  • 航线规划软件的使用（DJI Pilot、DJI Terrain Follow等）
  • 航向重叠率/旁向重叠率的参数设置依据
  • 不同地形（平原、丘陵、山地）的航线策略差异
  • 飞行高度、地面分辨率（GSD）与相机参数的关系
🎬 视频要点：通过不同地形案例，讲解航线参数的灵活调整逻辑
08
航拍技巧与影像获取
📺 25分钟 🎯 技能 📁 航拍摄影
运镜技巧 曝光控制 场景构图
→
核心知识点
  • 对焦模式、曝光补偿、白平衡的手动控制技巧
  • 五种经典运镜手法：前进/后退/侧移/旋转/俯冲
  • 不同光照条件下的飞行拍摄策略（顺光/逆光/晨昏）
  • ND滤镜的使用时机与效果
  • 航拍素材的质量判断：分辨率、色彩、清晰度、快速检查法
  • 飞行安全距离与障碍规避的目视判断
🎬 视频要点：大量航拍实飞片段演示，逐一拆解运镜动作的执行要点
09
影像数据处理与三维建模
📺 25分钟 🎯 技能 📁 数据处理
Pix4D ContextCapture DOM/DEM
→
核心知识点
  • 主流建模软件对比：Pix4Dmapper、ContextCapture、Metashape
  • 航测数据处理全流程：原始数据→对齐→刺点→空三→重建→输出
  • DOM（数字正射影像图）与 DEM（数字高程模型）的区别与用途
  • 三维模型精度评估指标：GSD、点云密度、绝对/相对精度
  • 常见建模失败原因与质量检查方法
  • 成果格式转换与行业软件对接（CAD、GIS、BIM）
🎬 视频要点：完整演示一个实际建模项目从导入到输出的全过程
10
多光谱与红外热成像技术
📺 25分钟 🎯 技能 📁 传感载荷
多光谱 热成像 NDVI指数
→
核心知识点
  • 多光谱/高光谱成像原理与波段的农业/环境意义
  • NDVI（归一化植被指数）的计算方法与判读逻辑
  • 红外热成像原理：温度场分布与设备检测应用
  • 多光谱/热红外数据采集的飞行参数设置要点
  • 专业分析软件的使用（TASSEOMATIC、DJI Terra等）
  • 多源数据融合：可见光+热红外+多光谱的联合判读
🎬 视频要点：同一地块同时采集三种数据，对比展示各自的应用价值
MODULE 04
行业实战 — 五大领域的真实战场
深入测绘地理信息、农业植保、电力巡检、应急救援、阵列表演五大行业，以真实项目串联技术与场景
11
测绘地理信息行业应用
📺 25分钟 🎯 行业实战 📁 测绘GIS
1:500地形图 实景三维 不动产权籍调查
→
核心知识点
  • 无人机测绘在1:500、1:1000地形图生产中的标准化作业流程
  • 实景三维中国建设政策与的市场空间分析
  • 不动产测绘中的无人机权籍调查应用
  • 水库、堤坝、河道等水利设施的无人机监测方案
  • 地形图精度验收标准与质量控制要点
  • 测绘资质要求与无人机测绘项目投标要点
🎬 案例串联：以某县1:1000地形图测绘项目为线索，完整展示从投标到成果交付
12
农业植保行业应用
📺 25分钟 🎯 行业实战 📁 智慧农业
变量施药 仿地飞行 农业遥感
→
核心知识点
  • 植保无人机的选型：载重、喷洒系统、作业效率的匹配
  • 航线规划与仿地飞行（雷达/RTK高度跟随）技术
  • 变量施药技术：基于多光谱影像的精准喷洒决策
  • 作业参数（高度、速度、流量、喷幅）与防治效果的关系
  • 小麦、水稻、柑橘等不同作物的作业方案差异
  • 农业无人机的运营模式：单机合作社、共享平台、大疆农业T50案例
🎬 案例串联：以某水稻主产区的统防统治项目，串联从病虫害监测到施药验收的全流程
13
电力巡检行业应用
📺 25分钟 🎯 行业实战 📁 电力能源
输电线路 缺陷AI识别 精细化巡检
→
核心知识点
  • 输电线路无人机巡检的两种模式：通道巡检 vs 精细化巡检
  • 杆塔航线自动生成与RTK精准定位技术
  • 可见光照片+AI缺陷识别系统（断股、锈蚀、异物、松脱）
  • 红外热成像在电力设备缺陷检测中的应用
  • 配电网（杆式变压器、柱上开关）巡检的特殊挑战
  • 国网/南网无人机巡检规范解读与数字化验收
🎬 案例串联：以某±800kV特高压直流线路巡检项目，展示从外业采集到缺陷报告全流程
14
应急救援与公共安全应用
📺 25分钟 🎯 行业实战 📁 应急救援
搜救 火情监测 堰塞湖 重大活动保障
→
核心知识点
  • 无人机应急救援装备体系：可见光/热红外/照明/喊话/抛投
  • 山岳搜救场景的无人机搜索策略（网格搜索、立体搜索）
  • 森林草原火情的无人机火场监测与态势标绘
  • 地震、洪涝等自然灾害中的无人机应急测绘
  • 大型活动低空安保的无人机部署与管控
  • 应急管理部、消防救援队伍无人机配备标准解读
🎬 案例串联：以某山岳救援真实案例，串联搜索、定位、引导、评估四大应急场景
15
无人机阵列表演
📺 25分钟 🎯 行业实战 📁 灯光秀
集群控制 RTK定位 节目编排 文旅演艺
→
核心知识点
  • 无人机阵列表演行业概述：市场规模、商业模式、主要应用场景
  • 飞行控制系统：TDOA定位技术与地面站同步机制
  • 单机操控 vs 阵列集群的差异与协同逻辑
  • 节目编排与3D建模：如何设计立体动态图形
  • 安全管控：电磁干扰、气象条件、应急处置方案
  • 空域申请流程与行业资质要求
🎬 案例串联：以国庆庆典、春晚、大唐不夜城等震撼灯光秀为线索，拆解技术实现全流程
MODULE 05
综合提升 — 从学员到从业者
以综合案例串联全部技能树，同时补充职业发展路径与行业资源对接，培养完整的从业者思维
16
综合实训项目与职业发展
📺 25分钟 🎯 综合实训 📁 职业闭环
综合项目 职业路径 资质认证 行业资源
→
核心知识点
  • 综合实训设计：选择真实场景，完成"需求分析→方案制定→外业飞行→数据处理→成果输出"全流程
  • 如何撰写一份专业的无人机应用项目技术方案
  • 从业路径规划：飞手→技术负责人→项目经理→创业
  • 必备证书体系：AOPA/CAAC执照、行业认证、安全员认证
  • 行业资源平台：政府采购平台、政企合作渠道、行业协会资源
  • 学员成果展示与互评，复盘课程全链路收获
🎬 课程收尾：以一个完整行业项目投标文件为成果输出，让学员带走一份可用的真实材料
